# Enterprise-Network-Design-and-Deployment-
A comprehensive Cisco Packet Tracer project for "Mentora Nexus," a 3-branch enterprise network by Haim Levhar. Features OSPF &amp; BGP routing , IPsec/GRE tunnels , AAA security , and HSRP redundancy. Also includes DNS, DHCP, HTTP, FTP, NTP, Syslog, and Mail services .
